<Update label="2026-04-17" description="Security hardening and navigation updates" tags={["Security", "Update"]}>
  ## Security hardening and navigation updates

  ### Updates

  - **Documentation link in header** — The site header and mobile menu now link directly to the [documentation](https://docs.sonicverse.eu) for quicker access to guides and references.
  - **Contact page in navigation** — The main navigation now includes a direct link to the contact page, making it easier to get in touch.

  ### Security fixes

  - **Contact form XSS protection** — All contact and confirmation email templates now sanitize user input, preventing cross-site scripting (XSS) via form submissions.
  - **Security headers** — The site now sends a strict Content Security Policy, HSTS, X-Frame-Options, Referrer-Policy, and Permissions-Policy headers on every response, reducing the risk of common web attacks.
</Update>

<Update label="2026-04-14" description="Security reporting, contact form improvements, and visual refresh" tags={["Feature", "Update"]}>
  ## Security reporting page, contact form improvements, and visual refresh

  ### New features

  - **Security vulnerability reporting page** — You can now report security vulnerabilities directly from [sonicverse.eu/security](https://sonicverse.eu/security). The page includes a submission form and outlines our responsible disclosure policy.
  - **Animated blog header** — The blog page now features an animated grid pattern in the header for a more polished look.

  ### Updates

  - **Redesigned footer** — The site footer has been restructured with improved navigation groups, a dynamic copyright year, and a direct link to our security policy.
  - **Improved contact form** — The contact and vulnerability report forms now include rate limiting and clearer error messages, so you get better feedback when submitting.
  - **Better accessibility** — Forms across the site now include proper ARIA attributes and improved keyboard navigation support.
  - **Typography refresh** — Headings, line heights, and letter spacing have been fine-tuned across the site for improved readability. Smooth scrolling is now enabled site-wide, with reduced motion preference support for users who need it.
  - **Updated security contact** — Vulnerability reports are now routed to security@sonicverse.eu for faster triage.
</Update>
