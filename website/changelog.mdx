<Update label="2026-04-06" description="v0.29.5 – v0.29.8" tags={["Release"]}>
  ## Security header hardening

  A series of releases adding production security headers to all routes.

  ### New features

  - **Content Security Policy** — Sitewide CSP header with nonce-based inline script and style blocking.
  - **HSTS header** — Strict-Transport-Security header for production routes.
  - **Env-driven config** — Explicit environment booleans for toggling CSP and HSTS.

  ### Bug fixes

  - Removed `X-Powered-By` header from Next.js responses.
  - Tightened CSP directives for scripts and styles.

  **Full changelog**: [v0.29.5...v0.29.8](https://github.com/sonicverse-eu/website/compare/v0.29.5...v0.29.8)
</Update>

<Update label="2026-04-06" description="v0.27.0" tags={["Release"]}>
  ## MDX content migration and site improvements

  Major release migrating all site content to local MDX, removing CMS dependencies, and refining the public-facing experience.

  ### New features

  - **MDX content migration** — All site content now lives in the repository as MDX files. No external CMS or Prismic dependencies.
  - **GitHub issue integration** — Contact form now creates GitHub issues for bug reports and feature requests.
  - **Slack community link** — Added Slack invite to contact page and site configuration.
  - **Updated project index** — Added Audiostreaming Stack, removed deprecated projects, and refined site messaging.
  - **Enhanced page layout** — Background glow effects and improved metrics row interactions.

  ### Bug fixes

  - Updated contact email address in site configuration.
  - Updated reporting email address in Code of Conduct.

  **Full changelog**: [v0.27.0](https://github.com/sonicverse-eu/website/releases/tag/v0.27.0)
</Update>
